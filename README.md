# LinComms

> **Invite-only communications platform** with **secure messaging**, **admin-controlled access**, and **role-based invites**.  
> Built with **Next.js, Node.js, Prisma, PostgreSQL, TailwindCSS**, and deployed via **Vercel**.

## 🚀 Features
- 🔒 Invite-only access model — only admins can invite new members
- 🧑‍🤝‍🧑 Role-based access control (Admin / Member)
- 💬 Real-time secure messaging with WebSockets
- 📱 Modern responsive UI with TailwindCSS
- 🐳 Dockerized for local development
- ✅ Tested with Jest + Cypress

---

## 🛠️ Built With
- **Frontend:** React, Next.js, TailwindCSS
- **Backend:** Node.js, Express, Socket.IO
- **Database:** PostgreSQL, Prisma ORM
- **Real-time:** WebSockets, Redis
- **DevOps:** Docker, Railway, Vercel
- **Testing:** Jest, Cypress
- **Auth & Security:** JWT Authentication, Role-based Access Control
